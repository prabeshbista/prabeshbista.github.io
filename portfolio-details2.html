Optical table's Cover & LabVIEW Program

The optical components mounted on the optical table remains stable with minimum deflection so that the alignment of laser path is unchanged. To prevent optical components from dust, and stray light , a cover box is needed. I designed and assembled the cover box for the optical table. I used SOLIDWORKS for designing.

LabVIEW is the commonly used graphical programming to control the hardware. The powermeter when connected to the computer needs software to display the output. I used LabVIEW to develop the program for Thorlabs PM100D powermeter.You can view my code at here.
